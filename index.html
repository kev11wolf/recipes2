About Me

This simple Meal Planner was created to help with weekly meal organization. By using drag-and-drop functionality, easily editing recipes, and generating a compiled grocery list, planning has never been simpler. Enjoy!

How To Use

  1. Make sure your "data/recipes" folder has a "recipesIndex.json" containing an array of recipe files.
  2. Each recipe JSON should have "id", "title", "ingredients" (array), "instructions" (string).
  3. When the page loads, it fetches each recipe JSON and displays them as draggable pills in the sidebar.
  4. Double-click any recipe pill to edit its details.
  5. Drag a recipe pill onto a day in the calendar to assign that recipe.
  6. Generate a grouped grocery list (ignoring measurements) by clicking "Download Ingredients List".
  7. Use "Download Meal Plan" to save your assignment choices; later "Upload Meal Plan" to restore them.
